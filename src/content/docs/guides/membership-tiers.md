---
title: Membership Tiers
description: How to configure membership levels for your network with custom dues amounts and durations.
---

Membership Tiers let you define the levels available in your network — such as Basic, Standard, and Premium — each with its own annual dues amount and benefits.

You'll find them in the sidebar under **People > Membership Tiers**.

## Creating a Tier

1. Click **Add Membership Level**
2. Enter a **name** (e.g., "Gold Member")
3. Set the **annual dues** amount
4. Set the **duration** in months (typically 12)
5. Set the **sort order** to control the display position
6. Save

## Notes

- Tiers are organization-specific — each org defines its own
- Changing a tier's dues amount does not retroactively affect existing memberships
- Deleting a tier is only possible if no active memberships reference it
