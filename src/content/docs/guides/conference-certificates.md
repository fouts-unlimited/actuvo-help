---
title: Conference Certificates
description: How to print attendance certificates for conference registrants.
---

Conference certificates are issued to registered attendees as proof of participation.

**Location:** Admin → Printing → Certificates (`/admin/printing/certificates/`)

## Print Conference Certificates

1. Navigate to **Admin → Printing → Certificates**
2. Select the **Conference** from the dropdown
3. Optionally filter by registration status (Confirmed, Attended, etc.)
4. Check the attendees you want to print certificates for
5. Click **Print Selected** to generate PDFs

## Notes

- Conference certificates use the conference name as the title line automatically.
- Each certificate includes the attendee's full name and the conference dates.
- This page is separate from the membership certificate page — it uses the conference registration system, not the membership system.
- You do not need to create a Certificate Type for conference certificates — they use a built-in template.
